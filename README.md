🎞️ Slideshow – Слайд-шоу на 
Интерактивное слайд-шоу для терминала с поддержкой папок, настраиваемой скоростью, случайным порядком и эффектами перехода.
Просматривайте изображения в виде ASCII-арта или цветных блоков, управляйте клавиатурой, наслаждайтесь анимацией!
Реализовано на 7 языках программирования с единым интерфейсом командной строки.

🚀 Возможности
Показ изображений – все файлы в указанной папке (или переданные как список).

Настройка скорости – регулируйте задержку между слайдами (опция -d, --delay).

Случайный порядок – перемешивание изображений (-s, --shuffle).

Эффекты перехода – плавное появление (fade) или мгновенный переход (по умолчанию).

Полноэкранный режим – масштабирование под размер терминала.

Информация о файле – отображение имени, номера, размера.

Управление клавишами:

← / → – предыдущее/следующее изображение.

Space – пауза/продолжить.

+ / - – увеличение/уменьшение скорости.

s – переключить случайный порядок.

f – переключить режим (цвет / ASCII).

r – перезапустить слайд-шоу с начала.

q – выход.

Цветной вывод – поддержка 24-битных цветов в терминале.

Кроссплатформенность – работает на Windows, Linux, macOS.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> <папка_или_файлы> [опции]
Аргументы
Аргумент	Описание
<папка>	Папка с изображениями (все файлы будут показаны)
<файлы>	Список файлов через пробел (если указаны, папка игнорируется)
Опции
Опция	Описание
-d, --delay <N>	Задержка между слайдами в секундах (по умолчанию 3)
-s, --shuffle	Показывать изображения в случайном порядке
-a, --ascii	Использовать ASCII-символы вместо цветных блоков
-r, --recursive	Рекурсивно обходить подпапки
-h, --help	Справка
Примеры
bash
# Показать все изображения в папке Pictures с задержкой 5 секунд
python slideshow.py ~/Pictures -d 5

# Случайный порядок с ASCII-режимом
python slideshow.py ~/Photos -s -a

# Рекурсивный обход папки и подпапок
python slideshow.py ~/Gallery -r
Управление
Клавиша	Действие
← / →	Предыдущее/следующее изображение
Space	Пауза / продолжить
+ / -	Увеличить / уменьшить скорость (задержку)
s	Переключить случайный порядок
f	Переключить режим (цвет / ASCII)
r	Перезапустить слайд-шоу (с начала)
q	Выход

🧠 Логика работы
Сбор файлов – сканирование папки (рекурсивно при -r) или использование переданного списка.

Фильтрация – только изображения с расширениями .jpg, .jpeg, .png, .gif, .bmp, .tiff, .webp.

Отображение – каждый слайд конвертируется в ASCII-арт или цветные блоки с учётом размера терминала.

Переходы – эффект fade (плавное появление) или мгновенный переход.

Управление – обработка клавиш в интерактивном режиме.

Таймер – автоматический переход к следующему слайду по истечении задержки.

✨ Дополнительные фичи
Адаптивное масштабирование – изображения подгоняются под размер терминала.

Информационная панель – показ имени файла, номера слайда, общего количества, текущей скорости.

Сохранение состояния – при перезапуске случайный порядок обновляется.
